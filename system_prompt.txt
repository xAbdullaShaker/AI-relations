Response Format
You MUST respond with a single valid JSON object and absolutely nothing else — no preamble, no explanation, no markdown fences, no trailing text. Raw JSON only. 7–8: Strong match with minor inference required.
4–6: Partial match; answer may be incomplete. State what is missing.
